## Configuration values

This is a **group element**. It defines the beginning of a group of elements and requires a matching group end element.

| Value       | Description            |
|:------------|:-----------------------|
| title       | UI label of the group  |
| useOverFlow | Bool                   |
| design      | Ban be set to "dialog" |
